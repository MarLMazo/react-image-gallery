# Photo Gallery React Application
An Image Gallery using React JS with Material UI. It has lazy loading and responsive to different mobile sizes.

![ReactGallery](/Image/react-gallery.webp)

## Technology/Library Use
1. ReactJS
2. Material-UI
3. API for Images - [Lorem Picsum](https://picsum.photos/)

## How to use/run the project

In your Terminal, type:
    
    npm install
To install all dependencies

    npm start
To run the program

The Port Use for this project is `3000`
